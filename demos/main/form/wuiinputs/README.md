# WUIForm - WUI inputs demo

## Description

This demo shows the use of WUIForm with fill style and WUI inputs components.

## WUI JS Stack

- WUIIcon - 0.2
- WUIForm - 0.4
- WUISelectpicker - 0.4
- WUIDatepicker - 0.4
- WUITimepicker - 0.4
- WUIColorpicker - 0.4
- WUISwitch - 0.5
- WUIIntencity - 0.3
- WUIButton - 0.4
